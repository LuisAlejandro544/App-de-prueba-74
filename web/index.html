Configuración de Google Play App Signing

Sincroniza tus firmas con la consola oficial. Genera comandos listos para la herramienta oficial PEPK con un clic para encriptar tu llave de firma existente y transferirla a Google.

Paso a Paso: Encriptación con PEPK

1

Preparar tu Keystore Existente

Identifica el alias de la llave y el archivo físico `.keystore` o `.jks` que deseas preparar para Google Play.

2

Componer Comando PEPK Personalizado

Ingresa tus datos a la derecha para formular el comando exacto para tu terminal:

java -jar pepk.jar --keystore=my-keystore.jks --alias=upload --output=encrypted_key.pepk --encryptionkey=eb10c44155190981222bde1fbb3e3170560b11c08bc0d302 --algorithm=RSA
3

Subir el archivo .pepk resultante

Ejecuta el comando en tu terminal e importa el archivo encriptado `encrypted_key.pepk` en Google Play Console para activar la firma en la nube.
